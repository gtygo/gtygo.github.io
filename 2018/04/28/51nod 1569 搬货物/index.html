51nod 1569 搬货物

发表于 2018-04-28 | 分类于 51nod题解

现在有n个货物，第i个货物的重量是 2wi 。每次搬的时候要求货物重量的总和是一个2的幂。问最少要搬几次能把所有的货物搬完。

样例解释：

1，1，2作为一组。

3，3作为一组。

输入
单组测试数据。
第一行有一个整数n (1≤n≤10^6)，表示有几个货物。
第二行有n个整数 w1,w2,…,wn，(0≤wi≤10^6)。
输出
输出最少的运货次数。
输入样例
样例输入1
5
1 1 2 3 3
输出样例
样例输出1
2
