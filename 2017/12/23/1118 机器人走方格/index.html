51nod 1118 机器人走方格

发表于 2017-12-23 | 分类于 51nod题解

M * N的方格，一个机器人从左上走到右下，只能向右或向下走。有多少种不同的走法？由于方法数量可能很大，只需要输出Mod 10^9 + 7的结果。
输入
第1行，2个数M,N，中间用空格隔开。（2 <= m,n <= 1000)
输出
输出走法的数量。
输入样例
2 3
输出样例
3
