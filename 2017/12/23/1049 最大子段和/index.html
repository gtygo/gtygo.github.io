51nod 1049 最大子段和

发表于 2017-12-23 | 分类于 51nod题解

N个整数组成的序列a[1],a[2],a[3],…,a[n]，求该序列如a[i]+a[i+1]+…+a[j]的连续子段和的最大值。当所给的整数均为负数时和为0。
例如：-2,11,-4,13,-5,-2，和最大的子段为：11,-4,13。和为20。
输入
第1行：整数序列的长度N（2 <= N <= 50000)
第2 - N + 1行：N个整数（-10^9 <= A[i] <= 10^9）
输出
输出最大子段和。
输入样例
6
-2
11
-4
13
-5
-2
输出样例
20
